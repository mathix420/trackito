# Trackito

Ultimate code tracker system.


## Development guide

### Get started

Require npm/yarn workspaces.

Make sure to have lerna installed globally.
```bash
npm i -g lerna
```

```bash
git clone https://github.com/mathix420/trackito.git
cd trackito
lerna bootstrap
```
